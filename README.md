# Vite+React 前端项目启动器

## 启动

```bash
yarn
```

## 介绍

本项目是一个基于 Vite+React 的前端项目启动器，包含了一些常用的功能，如：

- [x] React
- [x] React Router 路由
- [x] Recoil 状态管理
- [x] Axios 网络请求
- [x] Less CSS 预处理器
- [x] Eslint + Prettier 代码规范
- [x] Husky + Lint-Staged 代码提交规范

在你安装了自己的 UI 组件库后，就可以直接写页面了，不需要再配置这些烦人的 settings 了。
